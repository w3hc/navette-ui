# Navette

Bridge USDC from Sepolia to OP Sepolia.

- Live demo: **https://navette-ui.netlify.app**
- Navette API: https://github.com/w3hc/navette-api
- Swagger UI: https://navette.jcloud.ik-server.com/api

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```
cp .env.example .env
```

Add your own keys in the `.env` file (you can get it in your [Wallet Connect dashboard](https://cloud.walletconnect.com)), then:

```bash
pnpm dev
```

## Versions

- pnpm `v8.7.5`
- node `v20.9.0`
